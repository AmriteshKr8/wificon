# wificon
user friendly python interface for nmcli.
## Install guide:
#### get the source:
`git clone https://github.com/AmriteshKr8/wificon.git`
#### get into the directory:
`cd wificon`
#### make scripts executable:
`sudo chmod +x *`

### Install:
`./install`

### Uninstall:
`sudo rm -rf ~/.local/bin/wificon*`
